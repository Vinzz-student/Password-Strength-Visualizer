# Password Strength Visualizer

A modern, real-time password strength analysis tool built with React + Vite. Visualizes password patterns, estimates crack times, and provides security recommendations.

## 🚀 Features

- **Real-time Analysis**: Instant feedback as you type
- **Pattern Visualization**: See distribution of lowercase, uppercase, numbers, and symbols
- **Hack Time Estimation**: Realistic estimates of how long it would take to crack the password
- **Security Recommendations**: Actionable tips to improve password strength
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Beautiful UI**: Modern glass-morphism design with smooth animations

## 🛠️ Technologies Used

- React 18
- Vite
- Tailwind CSS
- Framer Motion (animations)
- zxcvbn (password strength estimation)
- React Icons
